What's New: Scheduled Focus Mode, Smarter Rooms, and the Mac App Is Coming

P
Peter Hoang March 26, 2026 · 5 min read

This one has been building for a while. Every session I've spent on Layers lately has been fixing the things that quietly bugged me while using the app, there are many changes, but below are the main.

v3.8 is a focused update. No new headline feature to put on the website. Just a lot of things that now work better than they did.

Scheduled Focus Mode and Internet Lockdown

This was the most-requested thing I'd heard from teachers. "Can Layers just automatically turn on Focus Mode at the start of school and after school hours?" Yes — now it can.

In the Classroom Sync panel there's now a Schedule section at the bottom. You set a start and end time for Focus Mode and/or Internet Lockdown — separately, independently. Layers checks every 60 seconds and enforces whatever the schedule says. It handles overnight ranges, so if you want lockdown from 10pm to 7am for a boarding school situation, that works too.

// How scheduling works
  • Two independent schedules — Focus Mode and Internet Lockdown each have their own start/end time. Setting one doesn't affect the other.
  • Survives teacher disconnect — if you close the app or the session ends, the schedule keeps enforcing on student devices until its end time or until a student explicitly leaves the room.
  • Persists across browser restarts — students' Chrome extension stores the schedule locally. If they restart their laptop at 8am and class starts at 9am, it still kicks in at 9am.
  • Shared between teachers — if two teachers share a room (e.g. homeroom + subject teacher), Teacher B sees whatever Teacher A already scheduled when they connect. Firebase is the source of truth.

The Chrome extension needed its own update to support this — that's v1.3.5, which should auto-update in Chrome within 24 hours.

A Cleaner Classroom Sync Panel

The Sync panel was getting crowded. Saved rooms used to appear as a row of small buttons that wrapped awkwardly. PIN Lock was buried in its own expandable section. The panel couldn't scroll, so once you had a few features enabled the whole thing felt cramped.

Three changes here in v3.8:

// Sync panel changes
  • Saved rooms are now a dropdown. Select a saved room to connect instantly — no password re-entry. A × button removes the selected room. "✨ New Room" is always the first option.
  • PIN Lock moved to room creation. When you create a new room you can optionally set a 4-digit PIN. Students can't disable Focus Mode without it. The Sync panel no longer has a separate PIN section.
  • The panel scrolls. All cards — Connection, Focus, Lockdown, Push Link, Block Sites, Schedule — sit inside a scrollable area. The panel stays a fixed size; the content adapts.

A Few Other Things

Room Creation Is Faster

There was a bug where creating a new room caused a 5-second delay before the app connected. It was caused by a Firebase write that was happening too early — before the teacher session was properly established. Fixed. Room creation is now instant.

Layers Talk — Continuous Listening and Meeting Minutes

Layers Talk v1.5 shipped a few weeks ago with a full set of new features — continuous listening mode (no more holding F5/F6), a live scrollable transcript, editable speaker names, HTML meeting minutes export, and a customizable toolbar. If you haven't updated, grab the latest from Gumroad. Existing customers get it free.

The Mac App Is Real

I've been asked about a Mac version since I launched Layers. The honest answer for a long time was: I'm on Windows, I don't have a Mac, and building cross-platform without owning the hardware felt like a wall.

That wall came down. I built a complete Mac version of Layers — Electron + React, floats above Keynote and Safari in full-screen, real microphone capture, Firebase Classroom Sync, noise meter, schedule, everything. The build pipeline runs on GitHub's free Mac servers, so I don't need to own a Mac to ship a Mac app.

It's not out yet. The first build is compiling. I still need to test it properly, sort out the download page, and decide whether to charge for it separately or bundle it with the Windows license. But it's real, it exists, and it will be available at thelayersapp.com when it's ready.

If you want to be notified the moment the Mac app drops, email me at questions@thelayersapp.com with the subject "Mac — notify me" and I'll let you know.
