Set up Guard Family

Follow these four steps to install protection on your child's device and connect it to your parent dashboard. The entire setup takes about 10 minutes.

🛡️ 5-app bundle · Up to 7 devices · No subscription
1

Set up child's device

⏱ ~3 minutes

Install the Guard agent on the device your child uses. Find your device type below and follow the steps.

🖥️ Windows PC
1

Download Guard Desktop

Copy LayersGuardDesktop.exe from the USB or Google Drive folder to your child's Windows PC and run it. The app runs silently in the system tray.

⚠️ Windows SmartScreen may block the first launch. Click "More info" → "Run anyway". This only happens once.
2

Write down the pairing code

Guard Desktop shows a pairing code (e.g. LG-A7X3) on first launch. Write this code here — you'll enter it in Step 3.

Write your code here
LG-_ _ _ _
✓
🖥️
Guard Desktop is running in the background

It blocks sites via DNS, monitors apps, enforces time limits, and reports to your dashboard — all silently.

🍎 Mac
1

Install Guard Mac

Copy the Layers Guard .dmg from the USB or Google Drive. Open it and drag the app into Applications. Launch it — a shield icon appears in the menu bar.

⚠️ macOS may warn about an unidentified developer. Go to System Settings → Privacy & Security and click "Open Anyway".
2

Write down the pairing code

Guard Mac shows a pairing code (e.g. LG-A7X3) on first launch. Write this code down for Step 3.

✓
🍎
Guard Mac is running in the menu bar

It blocks sites via DNS, monitors apps, enforces time limits, scans content, and reports to your dashboard.

📱 Android Phone
1

Install Guard Child

Copy LayersGuardChild.apk from the USB or Google Drive to your child's Android phone. Open it and tap Install. Follow the on-screen setup — it will ask for Accessibility and VPN permissions.

💡 These permissions let Guard monitor app usage and block harmful content. Guard does not read personal messages or passwords.
📲 To install APK files, you may need to enable "Install from unknown sources" in your phone's settings. Go to Settings → Apps → Special access → Install unknown apps and allow your file manager.
2

Write down the pairing code

After setup, Guard Child shows a pairing code (e.g. LG-A7X3) on screen. Write this code down for Step 3.

✓
📱
Guard Child is running

It monitors screen time, blocks apps, scans chat messages for harmful content, and enforces schedules — all in the background.

2

Set up parent access

⏱ ~2 minutes

Choose how you want to monitor your child's device. You can use both options — they connect to the same account.

📲 Option A: Phone App
1

Install Guard Controller

Copy LayersGuardController.apk from the USB or Google Drive to your own Android phone and install it. This is the parent app — it controls the child's device remotely.

4

Configure protection

⏱ ~3 minutes

Your dashboard gives you full control. Configure as many or as few features as you need.

⏱️

Daily Time Limit

Set max screen-time minutes per day. When time runs out, the device locks. Separate limits for weekdays and weekends.

Auto-enforced
🌙

Bedtime Lockout

Set hours when the device is fully locked (e.g. 22:00–07:00). All apps and internet blocked.

Schedule
🚫

Block Sites

Block domains via DNS — works across all browsers and apps. Add individual sites or use categories.

DNS-level
📵

Block Apps

Block specific apps on the child's device. Blocked apps close automatically. Set per-app time limits.

Per-app limits
🚨

Content Alerts

Get notified when harmful content is detected — violence, bullying, sexual content, or stranger-danger language.

Real-time
🔒

Emergency Lock

Instant
