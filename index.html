// Features

Everything you need
in the classroom.

🎙

Push-to-Talk Translation

Hold F5 to speak. Your words are transcribed and translated in real-time using Google Speech Recognition — no API key required.

文A

Floating Caption Window

A semi-transparent overlay floats above your screen so students can read your translated speech at any time during the lesson.

✏

Screen Drawing & Annotation

Draw directly over any application — pen, highlighter, line, arrow, rectangle, eraser. Undo with Ctrl+Z. Press Escape to exit.

⏱

Timer & Stopwatch

Countdown timer with presets (30s, 1m, 2m, 3m, 5m, 10m) and an alarm buzzer, plus a stopwatch mode with looping progress bar.

⚄

Random Student Picker

Slot-machine style animation picks a random student from a list of up to 100. Also opens PDFs, Word, PowerPoint, and Excel files directly.

⚠

Noise Meter

Live semicircular arc gauge (green/yellow/red) with adjustable sensitivity. An alarm fires when noise exceeds the threshold, with a 5-second cooldown.

🔒

PIN Lock System

Set a 4-digit PIN so students cannot disable Focus Mode, leave the sync room, or close the extension without teacher authorisation.

🚫

Custom Site Blocking

Add any domain to your blocklist and push it to all connected student Chromebooks in real time — on top of the 60+ default blocked sites.

📖

Multilingual Dictionary

Students highlight any word on any webpage and instantly look it up in their native language — right inside the toolbar, no new tab needed.

// Firebase Sync

Control every device
in the room.

The teacher's Windows app and student Chrome extensions communicate via Firebase Realtime Database. One command reaches every Chromebook in your classroom in under 3 seconds.

  • Teacher enters a room code — students join the same room
  • Focus Mode pushed to all students simultaneously
  • Block any site and push it live to every connected device
  • PIN lock prevents students from dismissing teacher controls
  • Live student count — see who is connected right now
  • Auto-disconnects after 45 minutes idle to save Firebase usage
