You'll see the Layers Guard icon in the top-right of your browser. Click it to open the Guard panel. Next, set your parent PIN to lock the settings.

2

Set PIN & block sites

⏱ ~1 min ✓ Done
▾
1

Set your parent PIN

Click the Guard icon → Options. You'll be prompted to create a 4-digit PIN. This PIN locks the settings so your child cannot change them.

💡 Remember this PIN — you'll need it every time you want to change Guard settings. There is no recovery option.
2

Add blocked sites

You have two options to add blocked sites:

⚡

Quick Setup

One-click import of ~60 curated sites

✏️

Add manually

Type domain names one by one

Click "Use Layers List" in the Guard options panel. A categorized list of ~60 common distraction sites appears with checkboxes. Select the categories you want to block, then save.

In the Guard options panel, type a domain name (e.g. youtube.com) into the input field and click Add. You can optionally set a schedule — e.g. block only from 08:00 to 21:00.

3

Set a lockout schedule (optional)

Enable Lockout to block all browsing during specific hours (e.g. 22:00–07:00 for bedtime). During lockout, every website is blocked — no exceptions.

✓

Sites are blocked

🚫

Blocked sites now redirect to a Guard page

When your child tries to visit a blocked site, they'll see a branded "Blocked" page instead. They cannot bypass it without your PIN.

3

Enable Content Protection

⏱ ~1 min ✓ Done
▾

Content Protection scans web pages for harmful text in real time — violence, bullying, sexual content, and stranger-danger language. It blurs harmful content before your child can read it. Everything runs on-device — no data leaves the browser.

1

Turn on Content Protection

In the Guard options panel, find the Content Protection section and toggle it ON. Choose your sensitivity mode:

Standard High-severity only (violence, sexual, stranger danger)
Strict Also catches bullying and medium-severity language
2

Choose which sites to scan

By default, Guard scans popular social media sites: Instagram, Facebook, Twitter/X, Reddit, TikTok, YouTube, Discord. You can add or remove sites from the scan list in the options panel.

✓

Content Protection is active

🛡️

Harmful content is blurred automatically

When Guard detects harmful text, the content is blurred and a warning overlay appears. The Guard icon shows a badge count of alerts. All scanning is on-device — private and fast.

🛡️ Guard Chrome is active!

Your browser is now protected. Blocked sites are redirected, harmful content is blurred, and all settings are locked behind your PIN.

1

Download Guard Desktop

Download the Guard Desktop installer and run LayersGuardDesktop.exe on your child's Windows PC. The app runs silently in the system tray — no visible window.

⚠️ Windows SmartScreen may block the first launch. Click "More info" → "Run anyway". This only happens once.
2

A pairing code appears

When Guard Desktop starts for the first time, it generates a pairing code (e.g. LG-A7X3) and shows it in a small dialog. Write this code down — you'll enter it on your parent device in Phase 3.

✅ Guard Desktop starts automatically when your child logs in. It runs in the background with no desktop icon — just a small shield in the system tray.
✓

Child's PC is ready

🖥️

Guard Desktop is running in the background

The agent will block sites via DNS, monitor app usage, enforce time limits, and report status to your parent dashboard — all silently.

1

Download Guard Mac

Download the Layers Guard .dmg file and drag the app into your child's Applications folder. Open the app — it lives in the menu bar as a small shield icon.

⚠️ macOS may show a warning that the app is from an unidentified developer. Go to System Settings → Privacy & Security and click "Open Anyway". This only happens once.
2

A pairing code appears

When Guard Mac starts for the first time, it generates a pairing code (e.g. LG-A7X3) and shows it in a dialog. Write this code down — you'll enter it on your parent device in Phase 3.

✅ Guard Mac starts automatically at login and runs silently in the menu bar. Your child won't see a dock icon or window — just a small shield in the top-right of the screen.
✓

Child's Mac is ready

🍎

Guard Mac is running in the background

The agent will block sites via DNS, monitor app usage, enforce time limits, scan for harmful content, and report status to your parent dashboard — all silently.

1

Install Guard Child

Install the Guard Child app on your child's Android phone. Open the app and follow the on-screen setup — it will ask for Accessibility and VPN permissions to enable protection.

💡 These permissions let Guard monitor app usage and block harmful content. Guard does not read personal messages or passwords.
2

A pairing code appears

After setup, Guard Child shows a pairing code (e.g. LG-A7X3) on screen. Write this code down — you'll enter it on your parent device in Phase 3.

✓

Child's phone is ready

📱

Guard Child is running

The app will monitor screen time, block apps, scan chat messages for harmful content, and enforce schedules — all in the background.

4

Configure protection

⏱ ~3 min
▾

Your dashboard gives you full control over your child's device. Here are the key features — configure as many or as few as you need.

⏱️

Daily Time Limit

Set a maximum number of screen-time minutes per day. When time runs out, the device locks and shows an overlay. Separate limits for weekdays and weekends.

Auto-enforced
🌙

Bedtime Lockout

Set hours when the device is fully locked (e.g. 22:00–07:00). All apps and internet are blocked. Different settings for weekdays and weekends.

Schedule
🚫

Block Sites

Add domains to the block list. The child device blocks them via DNS — works across all browsers and apps. Add individual sites or use categories.

DNS-level
📵

Block Apps

Block specific apps on the child's device. Blocked apps are closed automatically. Set per-app time limits to allow controlled access.

Per-app limits
🚨

Content Alerts

Get notified when harmful content is detected — violence, bullying, sexual content, or stranger-danger language. Alerts show the category and timestamp.

Real-time
🔒

Emergency Lock

Instantly lock the child's device from your phone or dashboard. Blocks all apps and internet immediately. Use when you need to get your child's attention.

Instant
