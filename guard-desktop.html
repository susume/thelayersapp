// Questions

Things parents ask us.

Can my child uninstall or disable the Windows app?
The agent runs in the background with no visible window. The tray icon is the only visible sign — and attempting to quit from it requires your parent PIN. The app also re-adds itself to Windows startup automatically every time it runs, so even if a child deletes the startup entry, the next launch restores it. Uninstalling requires admin access to the PC, which children typically don't have.
What happens if my child uses a VPN?
DNS-level blocking works through the hosts file, which redirects domain requests before they ever reach a VPN. For complete protection, you can also block VPN client applications directly — search for Mullvad, ExpressVPN, and other common VPN executables and add them to your blocked apps list.
Does my phone need to be online for the rules to work?
No. Rules are synced to the Windows agent and enforced locally. Once you've set up your blocks, schedules, and time limits, they continue working even if your phone has no internet connection. The live dashboard and remote commands require connectivity, but the protection itself never stops.
What does "block any browser" actually mean?
Most parental controls only filter inside Chrome. Layers Guard edits the Windows hosts file — a system-level file that redirects domain names to nowhere, regardless of which browser, app, or program is making the request. Chrome, Firefox, Edge, Brave, even desktop apps that access the web are all covered.
I forgot my parent PIN. What happens?
Open the Controller app on your phone, go to Settings → Security, and tap "Forgot PIN?" You'll be prompted to set a new one. The Windows agent uses the new PIN automatically — you don't need to touch the child's PC.
Can I manage more than one child / PC?
Yes. Each Windows PC generates its own device ID and pairs independently. From the Controller app you can switch between devices with a single tap. Each child gets their own rules, screen time data, and alert history.
